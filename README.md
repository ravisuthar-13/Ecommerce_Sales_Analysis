# Ecommerce_Sales_Analysis
Data analysis project showcasing Ecommerce sales analysis using python, MYSQL and power Bi.


Overview
This project provides a comprehensive end-to-end data analytics solution designed to extract actionable insights from e-commerce sales data. The workflow covers the entire data lifecycle: from raw data cleaning and exploratory data analysis (EDA) in Python to advanced relational querying in MySQL, culminating in an interactive Power BI dashboard and a professional presentation created with Gamma.

📁Dataset
The project utilizes a dataset containing 5,000+ records of e-commerce transactions.

Key Columns: order_id, order_date, customer_name, region, city, category, sub-category, product_name, quantity, unit_price, discount, sales, profit, and payment_mode.

Time Period: Covers transactions across 2024 and 2025.

🛠️Tools & Technologies
Python (Pandas, Scikit-Learn): Data loading, cleaning, and Exploratory Data Analysis (EDA).

MySQL: Relational database management and complex analytical querying.

Power BI: Data visualization and interactive dashboard creation.

Gamma AI: Professional presentation and reporting.

Jupyter Notebook: Environment for executing Python scripts.

Steps Involved
1. Data Cleaning & EDA (Python)
Handled missing values and standardized data types for consistency.

Performed statistical analysis to identify trends and outliers.

Exported the cleaned dataset to .csv and .sql formats for downstream use.

2. Database Management (MySQL)
Imported the cleaned data into a MySQL database.

Executed optimized queries to calculate business KPIs such as:

Top 5 customers by total spending.

Monthly revenue trends.

Profit margin percentages by region.

3. Data Visualization (Power BI)
Connected the cleaned data to Power BI.

Built an interactive dashboard featuring:

Sales vs. Profit Trends: Monthly performance tracking.

Geographic Analysis: Sales distribution across top cities and regions.

Product Performance: Identifying high-volume and high-profit categories.

4. Reporting & Presentation
Synthesized findings into a structured report.

Generated a professional slide deck using Gamma AI to communicate results to stakeholders effectively.

Dashboard
The Power BI dashboard focuses on key business metrics:

Total Revenue & Total Profit: High-level summary cards.

Average Order Value (AOV): Analysis of customer purchasing behavior.

Payment Mode Distribution: Insight into preferred customer payment methods.

Results & Insights
Identified the top-performing product categories and sub-categories.

Recognized specific regions that contribute significantly to the total profit margin.

Discovered seasonal sales peaks, allowing for better inventory and marketing planning.
